# Voting Application

A clean, professional, and easy-to-customize voting application built with HTML, CSS, and JavaScript.

## Features
- Vote for your favorite option.
- Real-time results display.
- Simple and intuitive design.

## How to Use
1. Clone the repository or download the files.
2. Open `index.html` in your browser.
3. Click the "Vote" button next to your preferred option.
4. View the results in real-time.

## Customization
- Add more options by duplicating the `.option` div in `index.html`.
- Modify the styles in `styles/styles.css` to match your branding.
- Extend functionality in `scripts/main.js`.
